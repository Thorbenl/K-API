# K-API
A small API which returns kpop artists, albums, etc.

Getting Started
---------------

```sh
# clone it
git@github.com:Thorbenl/K-API.git
cd K-API

# Install dependencies
npm install

# Start MongoDB

# Env Variables
Windows & Linux
SET kpi_jwtPrivateKey=hello

Mac
export kpi_jwtPrivateKey=hello

# Start server
npm run start
